# **App Name**: SVLSM Stock Manager

## Core Features:

- Inventory Display: Display the current inventory items with their selling prices.
- Admin Panel: Provide an admin panel to record stock input and output with authentication.
- Pin Authentication: Require a PIN for login authentication to secure the app.
- Door Gallery: A panel to upload and display images of doors made in-house.

## Style Guidelines:

- Primary color: Deep brown, evoking a woody theme.
- Secondary color: Light beige, providing a neutral backdrop for clarity and readability.
- Accent color: Forest green, for key actions like 'Add Stock' or 'Sell', drawing the user's attention.
- Font: 'Arial', sans-serif for clear, professional readability in both headlines and body text.
- Clean, tabular layout for inventory display, easy to navigate.
- Use icons for types of machinery or materials managed. For instance, a saw blade icon, or a wood plank icon.
- The theme should be total woody
- Use some wood texture images or sawn woods for decorations
